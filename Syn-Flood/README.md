# Syn Flood

Simple script for doing syn flood

## Getting Started

Grab the compiled file and launch it.

Or grab the c file and compile yourself with somethin like this:

```
gcc $(libnet-config --defines) -o synflood synflood.c -lnet
```

### Prerequisites

This scripts needs C installed and libnet installed

### Installing Dependencies

It depends on your OS:

Debian/Ubuntu: ``` sudo apt install libnet```

Arch: ``` sudo pacman -S libnet```

Windows: ``` Don't use it, use Linux ```

### Usage

Just execute the script as root and follow instructions if needed
